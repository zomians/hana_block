# DESIGN — 意匠カタログと図面の見せ方

「どんな柄を、どう見せるか」を集約する。意匠の正体・命名・タイリング構成はここで確定させる。

## 意匠カタログ

### 麻の葉（slug: `asanoha`）
- 表示名: **麻の葉（四つ組）**
- 内容: 対角格子による麻の葉柄。
- タイリング構成: **4つで1デザイン**。1ブロックは意匠の 1/4 で、2×2（鏡像配置）で柄が完成する。製作図は 2×2 で見せる。
- 展開サイズ:
  - 390 × 390 × 100 / 全壁厚 35 / 開口率 約34.5%
  - 190 × 190 × 100 / 全壁厚 20 / 開口率 約27.2%

### 組子（slug: `kumiko`）
- 表示名: **組子（麻の葉）**
- 内容: 組子の麻の葉。小さな麻の葉ユニットを1ブロック内に密にタイリングした、星が腕を共有して**連続**する格子意匠。麻の葉（`asanoha`＝大柄・四つ組）とはスケールと構成で区別する。
- タイリング構成: **単体完結**。1ブロック内で小麻の葉を反復し、それ単体で柄が完成する（asanoha の「4つで1デザイン」と構造で別物）。製作図は単体正面図を主役に、2×2配置図で接合を確認する。
- 構成（実測確定）: **正三角格子（60°・歪み0）＋各三角形の重心スポーク**。格子頂点＝星中心に6枚の葉（60°菱形 97.5×56.29）が連続。格子ピッチ L=97.5 / 行ピッチ h=84.44、**谷中心配置**（左右の縦線を周枠に整合）。
- 仕上げ: **上下＝星行を周枠で締める／左右＝織りを縦線で対称断ち**。上下は枠で締めてキリ良く、左右は 390=97.5×4。ブロックは H 非対称・V 対称なので、製作図の 2×2 配置は**左列を左右反転（鏡像タイル）**して継ぎ目の柄を連続させる。密度 約4×4（中）。左右の納まりは α' 採用（ADR-0002 再検討節）。
- 展開サイズ:
  - 390 × 390 × 100 / 全壁厚 12（格子壁・均一）/ 周枠 上下20・左右24.375 / 接合壁厚 上下40・左右48.75 / 開口率 約23%
- 寸法は基本設計図（`patterns/kumiko/kumiko-base.svg`）で実測確定済み。サイズ・密度・壁厚・周枠の決定根拠は ADR-0002。

## 図面の見せ方（共通レイアウト方針）
JIS 製図に倣った「図面シート」形式とする。コピー元は `template/template.html`（製作図はこれを複製して作る）。
- 用紙: A4 landscape。印刷して手渡せること。
- 図面枠: 二重枠＋四辺のセンタリングマーク。
- 構成要素: 上部に2図（① 基本設計図 ＋ ② 製作図〔意匠ごとの構成〕）、下部に表題欄。
- 表題欄（タイトルブロック）: マストヘッド（HANA·BLOCK／花ブロック 製作図）＋ スペック格子（意匠・寸法・縮尺・全壁厚・接合壁厚・開口率・材質・日付）。
- 必須表記: 寸法線 / 縮尺 / 表題欄の各値。
- 書体: 和文 Noto Sans JP、数値・記号・ラベルは IBM Plex Mono のペアリング。
- 配色: 紙白＋墨（ink）。朱色（accent）を①②インデックスと要点注記にだけ効かせる。
- 画面では極薄の方眼＋微粒子オーバーレイ、印刷時は白地・影なしで崩れないこと。
