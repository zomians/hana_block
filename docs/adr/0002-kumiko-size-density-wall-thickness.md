# ADR-0002: 組子（麻の葉）を 390×390×100 単一・密度中〜密（約5×5）・均一壁厚で起こす

- ステータス: 採用
- 日付: 2026-06-15

## 文脈
新規意匠「組子（kumiko）」を、タニハタの組子「麻の葉」のように小さな麻の葉ユニットが連続して並ぶ密フィールドとして作る。意匠の根幹（連続フィールド・単体完結）が固まり、サイズ・密度・壁厚を決める必要があった。製造元（山内コンクリートブロック）のカタログ `reference/catalog-250707.pdf` を物差しに検討した。

## 決定
- **サイズ: 390 × 390 × 100 の単一サイズ**。190/290 は採らない。
- **タイリング: 単体完結**（1ブロック内で小麻の葉を反復し、それ単体で柄が完成）。asanoha の「4つで1デザイン」とは構造で別物。
- **密度: 中〜密（麻の葉ユニット約 5×5）**。
- **壁厚: 均一**（asanoha の ADR-0001 を踏襲）。全壁厚は 15mm 級を目標とし、正確な全壁厚・開口率・接合壁厚は基本設計図 `patterns/kumiko/kumiko-base.svg` を描いて実測確定する。

## 理由
- カタログの意匠系花ブロックは厚100×390×390 が de-facto 標準で、確実に発注が通る。既存 asanoha-390 とも寸法が揃い、意匠軸で並ぶ。
- 390 角は細かい組子を入れるキャンバスの余裕があり、カタログに「いと（I-10）」「角4/5ミンサー（内部仕切り 12mm）」「グリッド（G-1）」など同等密度の組子・格子が実在する＝鋳造可能性が裏付けられる。190/290 では1ユニットが極小になり壁が薄すぎて密フィールドと相性が悪い。
- 密度 中〜密（約5×5・全壁厚 15mm 級）は「小さい麻柄がたくさん・連続」という意匠意図に最も忠実で、かつカタログの鋳造実績（仕切り 12mm）の内側に収まり安全。極細（6×6 以上・12mm 級）は強度・鋳造リスクが上がるため採らない。
- 均一壁厚は組子＝格子の性質に自然で、柄が整って見える（asanoha ADR-0001 と同じ理由）。
- 単体完結により意匠が1ブロックで完結し、製作図1枚で形状が一意に決まる（BRIEF 成功基準）。代案（小ブロックを施工で多数タイリングして柄を作る）は鋳造ブロックの境界・目地で麻の葉の連続が割れ、施工＝スコープ外に踏み込むため採らない。

## 影響
密度・壁厚を変えると開口率・接合壁厚・`kumiko-base.svg`・製作図がすべて変わるため、後戻りには各図面の作り直しが伴う（asanoha ADR-0001 と同様）。正確な全壁厚・開口率は base.svg 実測時に本 ADR の目標値を確定し、必要なら追記する。
