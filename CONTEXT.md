# CONTEXT — 用語

この案件で使う言葉の正準定義。図面・docs・コミットで言葉が揺れたらここに合わせる。

- **花ブロック** … コンクリート製の装飾（透かし）ブロック。本案件の製作対象。
- **パターン** … 製作する1種類のブロックを指す単位。**意匠 × サイズ × 壁厚** で一意に決まる。例: 麻の葉 / 390 / 壁厚35。
- **意匠** … 柄そのもの（形状デザイン）。サイズや壁厚とは独立。本案件では次の2つ:
  - **麻の葉（`asanoha`）** … 対角格子による麻の葉柄。表示名「麻の葉（四つ組）」。
  - **組子（`kumiko`）** … 組子風の格子意匠。新規追加予定。
- **基本設計図** … 単体ブロックの素図。壁厚・開口率など寸法を詰める最初の工程の成果物（例: `base.svg`）。
- **製作図** … メーカーに渡す最終図面。**タイリング構成（単体完結／2×2 など）は意匠による**ため、用語自体は構成非依存。
- **4つで1デザイン** … 麻の葉の性質。1ブロックは意匠の 1/4 で、4つを 2×2（鏡像配置）で並べると柄が1つに完成する。全意匠に当てはまるわけではない（→ 構成は DESIGN.md の意匠ごとに定義）。
- **全壁厚** … そのパターンの基本となる壁の厚み（例: 麻の葉390なら 35mm）。
- **接合壁厚** … ブロックを隣り合わせて積んだとき、隣接面で壁厚が合算される値（例: 20 + 20 = 40mm）。
- **開口率** … ブロック正面の面積に対する開口（抜き）部分の割合。
